# CineVault

A movie discovery platform built with React and Vite, powered by the TMDB API.

## Features

- Browse movies: Now Playing, Popular, Top Rated, Upcoming
- Filter by genre
- Movie detail pages with overview, rating, budget, release year
- Full cast listing with character names
- Trailers & video embeds from YouTube
- Pagination
- Wishlist and bookmark functionality

## Tech Stack

React, JavaScript, Redux Toolkit, React Query, Tailwind CSS, Vite, TMDB API
